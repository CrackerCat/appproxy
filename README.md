# appproxy

基于flutter和[tun2socks](https://github.com/xjasonlyu/tun2socks)
appproxy 是一个轻量级的VPN代理工具，支持HTTP, SOCKS5协议

# 依赖项目
- [tun2socks](https://github.com/xjasonlyu/tun2socks)
